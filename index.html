Procedurally Generated Waves - Individual

I had the most fun developing this project, which involved the procedural generation of waves, while also adding a mesh that allows collisions. There is also a mini-game in which you can shoot the orange cubes for points to progress to deeper levels. The purple cube's function is to demonstrate the colliders.

OpenGL Chess - Individual


Chess board, pieces, and terrain created with OpenGL. This project required plotting all vertices, positioning the objects once they're made, creating an import method and using it for textures, applying said textures, and finally creating the terrain and the skybox around it all. This is all done purely through C++.

Evolutionary Algorithms - Individual


Using a Unity car racing game provided by Vega, I created an algorithm that evolves the cars to reach further distances in lower amounts of time, all while avoiding local optimums.

Game Development and Math:


Math is beautiful. One of the benefits of being a game developer is the ability to see the visual effects of math in a virtual environment, and experiment with the many variations of different algorithms and expressions. An example is seeing Wolfram's Elementary Cellular Automata on the left, and the generating of cubes in Unity world space at points that are described by an algorithm, on the right.
